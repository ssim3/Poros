# Assembly Language

Because of the difficulty in reading Machine Code / Language, assembly language was created as a means for humans to program machine code.

Assembly language must be converted into Machine code before it can be run on a computer. This process is done through an assembler

# Low Level Languages

Machine languages and Assembly languages are considered Low Level Languages.

These languages are:
- Not portable (Runs on certain sets of machines but not others due to the difference in computer architecture)
- Requires deep knowledge of the architecture itself
- Very fast.


# High Level Languages

To address the previously stated issues, high level languages were created. C++ is a high-level language.

Programs written in high level languages need to be translated into machine code for computers to understand. There are two ways of doing this: Compiling and Interpreting.

High Level languages allows the programmer to write the program without much knowledge on the hardware or platform the program will be running on.
