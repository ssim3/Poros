# Line

The function of a line is described as follows:
$$
f(x) = mx + b
$$
- The constant m represents the slope of the line.
- The constant b represents the y-intercept.

Properties:
- Domain: $\mathbb{R}$. The function $f(x) = mx + b$ is defined for all real numbers.
- Image: $\mathbb{R}$ if $m\neq {0}$. If $m = 0$ the function is constant $f(x)=b$, so the image set contains only a single number b
- x = -b / m, the x-intercept of the function. Obtained by solving f(x) = 0
- The inverse function:
$$
f^{-1}(x) = \frac{1}{m} (x - b)
$$

# Polynomials

General function of polynomial function:
$$
f(x) = a_{0} + a_{1}x + a_{2}x^2 + a_{3}x^3 + \dots + a_{n}x^n 
$$
The constants $a_{i}$ are known as the coefficients of the polynomial.

A polynomial of degree n has n+1 coefficients a_0 ... a_n.

# Cosine

![[Pasted image 20260112210736.png]]

$$
f(x) = \cos(x)
$$

The cosine function is the sine functon shifted to the left by $\frac{\pi}{2}$ . 

Starts at cos(0) = 1, drops to cross the axis at x = $\frac{\pi}{2}$, this continues until reaching the minimum value of x = $\pi$. It then moves up again, crossing the x-intercept at $\frac{3\pi}{2}$ , and once again reaches its maximum value at $2\pi$

Domain: cos(x) accepts all inputs.
Image: { y e R | -1 <= y <= 1}
Roots: Multiples of $\frac{\pi}{2}$ , by $\pi$ 

Sin and Cosine relation: $\sin^2(x) +\cos^2(x)=1$

# Tangent

The tangent function is a ratio of the sine and cosine functions.

$$
\tan(x) = \frac{\sin(x)}{\cos(x)}
$$
